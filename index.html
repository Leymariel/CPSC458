Algorithmic Trading

The algorithmic trading project is a program I have been working on for a few months now. The program is designed to take in stock data which are used to compute numerous metrics such as garman klass volatility, rsi and macd to classify stocks into clusters based on similar performance. The algorithm creates an optimized portfolio for each month to assist a user on making stock market investments. The project served as a fun way of looking into the intricacies and volatility of the stock market and how to use data to make informed decisions.
